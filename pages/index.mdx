### How to contribute?

1. Go to GitHub repo

[https://github.com/vlhsmylv/docs](https://github.com/vlhsmylv/docs)

2. Clone the repo

```sh
git clone https://github.com/vlhsmylv/docs.git
```

3. Install dependencies

```sh
npm i
```

4. Create your branch

```sh
git switch -c <your-branch-name>
```

5. Make your changes

6. Push your changes

```sh
git push -u origin <your-branch-name>
```

7. Create a pull request

8. Wait for review

9. Merge your changes

10. Done!

_Thanks for the contribution!_
